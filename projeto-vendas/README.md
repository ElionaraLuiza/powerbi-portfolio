## Projeto Vendas - E-commerce

### Versão 1 - Dashboard Inicial

Este dashboard foi desenvolvido no início do curso de Power BI
com foco em aprendizado de visualizações e KPIs básicos.

### O que foi aplicado
- KPIs: faturamento, pedidos e itens vendidos
- Filtro por data e produto
- Gráfico de faturamento mensal
- Ranking por categoria
- Métodos de pagamento
- Análise por canal (Website x App)

### Objetivo desta versão
Treinar construção de dashboards e entendimento dos dados.

### Próximos passos
- Criar métricas de crescimento (%)
- Comparação mês a mês
- Criar aba de insights
- Melhorar layout e storytelling
